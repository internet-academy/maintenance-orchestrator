# Project Context: Personal Agents Framework

> **Status**: Upgraded - Technical Department Specialization Enabled

## Overview
A multi-agent orchestration system that enforces a rigid pipeline: **Architect (Contracting) -> Specialist (Implementation) -> Sentry (Audit)**.

## System Architecture
| Role | Responsibility |
|------|----------------|
| **Master Agent** | Orchestrates the mandatory verification pipeline. |
| **Architect** | Defines the `SUCCESS_CRITERIA.md`. |
| **Specialists** | Specialized departments (Pixel, Logic, Ops, Scout, Aero, Kaizen, Cart). |
| **Sentry** | Ruthlessly audits work against the contract. |

## Technical Departments
- **Pixel**: Frontend (Next.js, Tailwind v4).
- **Logic**: Backend (Django, Go, SQL).
- **Ops**: Infrastructure (Docker, Vercel, Automation).
- **Scout**: Security, Research, Compliance.
