# Project Context: Personal Agents Framework

> **Status**: Upgraded - Protocol of Strict Verification Enabled

## Overview
A multi-agent orchestration system that automatically enforces a rigid pipeline: **Architect (Contracting) -> Specialist (Implementation) -> Sentry (Audit)**. This ensures all deliverables meet a strict standard before being presented to the user.

## System Architecture
| Role | Responsibility |
|------|----------------|
| **Master Agent** | Orchestrates the mandatory verification pipeline. |
| **Architect** | Defines the `SUCCESS_CRITERIA.md` for every task. |
| **Specialists** | Implement the technical or creative requirements. |
| **Sentry** | Ruthlessly audits work and manages the feedback loop. |
